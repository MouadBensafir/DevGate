## Project Setup

### Installation
To install the project dependencies, run:
```
npm install
```

### Development
To start the development server with hot-reloading, use:
```
npm run serve
```

### Production and Deployment
To build the project for production and deploy it, run:
```
npm run build
firebase deploy
```
